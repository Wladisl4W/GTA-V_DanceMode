# 🕺 Dance Mode — GTA V Mod

Мод для GTA V, позволяющий добавлять NPC в список и запускать их танцевать синхронно.

## Требования

- **ScriptHookV** (основной)
- **ScriptHookVDotNet 3** (SHVDN3)
- **LemonUI SHVDN3**

## Установка

1. Скопируйте содержимое папки **`Ready To Use`** в корневую папку GTA V (где `GTA5.exe`).
2. Структура должна выглядеть так:
   ```
   GTA V/
   └── scripts/
       └── DanceMode.dll
   ```
3. **Готово!** Мод сам найдёт `FavouriteAnims.xml` из Menyoo!

## Управление

| Клавиша | Действие |
|---------|----------|
| **Y** | Открыть/закрыть меню |
| **Backspace** | Назад (в меню танцев) |
| **F5** | Отладочная информация |

## Как использовать

1. Наведите камеру на NPC.
2. Нажмите **Y** → **«Добавить педа»**.
3. Выберите **«Выбрать танец»** → понравившийся танец.
4. Наблюдайте!

## 💃 Танцы

Все танцы загружаются автоматически из **`FavouriteAnims.xml`** (избранные анимации из Menyoo).

### 🔍 Автоматический поиск файла:

Мод сам ищет `FavouriteAnims.xml` в следующих местах (по порядку):
1. `GTA V/scripts/FavouriteAnims.xml` (рядом с DLL)
2. `GTA V/menyooStuff/FavouriteAnims.xml` (стандартная папка Menyoo)
3. `GTA V/FavouriteAnims.xml` (корень игры)

**Ничего копировать не нужно!** Если у тебя есть Menyoo — мод сам найдёт файл с анимациями!

## 💾 Автосохранение

Мод **автоматически сохраняет** текущий танец и позиции педов при выходе из игры.

### Как это работает:

1. Вы добавили педов → выбрали танец → педы танцуют
2. Вышли из игры → мод создал автосохранение
3. Загрузили карту через Object Spooner → педы появились
4. Запустили игру → мод **нашёл педов по позициям** и **автоматически запустил танец**!

### Ручное управление:

| Действие | Как |
|----------|-----|
| Сохранить | Меню → **"Сохранить танец"** или **Ctrl+S** |
| Загрузить | Меню → **"Загрузить автосохранение"** |

Файлы сохранений: `scripts/DanceModeSaves/AutoSave.json`

## Сборка из исходников

Откройте `Source Code/DanceMode.csproj` в Visual Studio или через `dotnet build`:
```
cd "Source Code"
dotnet build --configuration Release
```
